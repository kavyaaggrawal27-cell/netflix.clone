# 🎬 Netflix Clone

A responsive Netflix landing page clone built using HTML and CSS.

## 🚀 Live Demo
🔗 https://kavyaaggrwal27-cell.github.io/netflix.clone/

---

## 📌 Features

- Netflix inspired UI
- Hero section with background image
- Trending movies horizontal scroll section
- FAQ accordion section
- Curved divider section
- Footer with grid layout
- Hover effects and transitions
- Fully deployed using GitHub Pages

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- GitHub Pages
